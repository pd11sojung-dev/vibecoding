# 비빌언덕 (Bibil-Eondeok) - 정적 웹 MVP

낯선 지역에서도 잠시 기대어볼 사람과 공간을 찾는 청년을 위한 관계 기반 지역 경험 매칭 서비스, **비빌언덕**의 정적 웹 MVP 프로토타입 프로젝트입니다.

---

## 1. 프로젝트 설명
서울 및 수도권 청년들이 지방 살이에 대해 느끼는 막연한 불안감(안전 문제, 외로움, 현실적인 정보 부족 등)을 해소할 수 있도록, **'이동 가능 시간'**과 **'멘토의 취향/일상 루틴'**을 중심으로 지역 경험을 안전하게 탐색하고 동행을 신청하는 사용자 흐름을 검증하기 위해 구현되었습니다.

---

## 2. MVP 범위
본 프로토타입은 프론트엔드 정적 UI(HTML5, CSS3, Vanilla JS)로 구성되어 있으며 다음 기능을 검증합니다.
* **출발지 및 관심 지역 설정:** 출발지 기준의 가상의 이동 범위 확인
* **관심사 태그 선택:** 자신의 취향에 맞는 키워드 선택 및 하이라이트
* **취향 기반 멘토 추천:** 3개의 겹쳐진 추천 카드 레이아웃과 예상 소요 시간 확인
* **멘토 일상 & 정보 탐색:** 멘토의 하루 루틴 타임라인과 현실적인 생활 적응 팁 확인
* **체험/동행 신청:** 원하는 일정 유형 선택 및 정적 일정표 조회를 통한 가상 신청 흐름 완결

### MVP 제외 범위 (정적 모형)
* 실제 회원가입/로그인 및 백엔드 데이터베이스 연동
* 실제 지도 API 및 위치 연동 기능
* 실제 결제 및 실시간 채팅, 알림 기능
* 실제 예약 확정 및 어드민 기능

## 5. 기술 스택
* **HTML5:** 시맨틱 웹(Semantic Web) 마크업 가이드 준수
* **CSS3:** 바닐라 CSS (외부 라이브러리, CSS 파일 분리 및 전처리기 사용 금지)
* **JavaScript:** 인터랙션(태그 활성화, 간단한 클릭 반응 등)을 제어하는 최소한의 Vanilla JS

## 7. 주요 화면 설명
1. **시작 화면 (`index.html`):** 비빌언덕의 비전 문구와 귀여운 중앙 로고 일러스트, 온보딩으로 이동하는 링크와 바로 시작하는 메인 CTA를 제공합니다.
2. **온보딩 화면 (`onboarding.html`):** 3가지 핵심 기능(시간 탐색, 취향 매칭, 살아보기 신청)을 명쾌한 요약 카드로 전달합니다.
3. **출발지/관심 지역 설정 (`neighborhood.html`):** 가상의 지도 일러스트와 이동 시간 슬라이더를 통해 접근성을 점검하는 정적 컨트롤을 구성합니다.
4. **관심사 선택 (`interests.html`):** `#텃밭`, `#도예체험` 등 14개 이상의 관심사 키워드를 클릭하여 다중 선택할 수 있는 동적 UI가 포함되어 있습니다.
5. **멘토 추천 카드 (`mentor-cards.html`):** 양옆의 카드가 비껴 누워있고 중앙의 양양 요가 멘토 카드가 시각적으로 돌출 강조되는 입체적인 3D 스타일 겹침 카드 레이아웃을 구현했습니다.
6. **멘토 일상 엿보기 (`mentor-day.html`):** 고성 도예 멘토의 시간대별 루틴을 타임라인으로 표현하고, 방문 난이도와 교통 꿀팁 등의 생활 밀착 정보를 제공합니다.
7. **동행 신청 (`apply.html`):** 반나절 체험부터 2주 살이까지의 플랜 테이블을 보여주고, 체험 완료 후 후속 연결을 안내하는 안내 창과 최종 확인 인터랙션을 제공합니다.
